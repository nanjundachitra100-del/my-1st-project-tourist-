Tourist Guide Web Application

Project Overview

The Tourist Guide Web Application is a web-based platform designed to help tourists explore destinations by providing useful and up-to-date information. Users can search for a tourist place and view details such as its history, description, and other relevant information retrieved from Wikipedia.

The application offers a simple and responsive interface, making it easy for users to discover information about different tourist attractions.

Features

- Search for tourist destinations.
- Fetches real-time information from Wikipedia using Google APIs.
- Responsive and user-friendly interface.
- Fast search results.
- Clean and simple design.

Technologies Used

Frontend

- HTML5
- CSS3

Backend

- JavaScript
- Node.js

APIs

- Google API (Free API Key)
- Wikipedia data source

Project Structure

- Frontend developed using HTML and CSS.
- Backend developed using Node.js and JavaScript.
- API integration to retrieve tourist information dynamically.

How It Works

1. The user enters the name of a tourist place.
2. The frontend sends the request to the Node.js backend.
3. The backend uses the configured Google API to retrieve relevant information from Wikipedia.
4. The fetched data is processed and displayed on the website in an easy-to-read format.

Installation

1. Clone the project repository.
2. Install dependencies using:
   npm install
3. Add your Google API key to the project configuration.
4. Start the server:
   npm start
5. Open the application in your browser.
